# Sabiduría de la Colmena - Comunidad

<p align="center"><img src="https://laragon/www/Sabidur-a_de_la_Colmena/public/images/colmena_logo.png" width="200" alt="Colmena Logo"></p>

Este proyecto consiste en una plataforma de comunidad para apicultores donde pueden compartir conocimientos, experiencias y consejos sobre apicultura.

## Características

- **Publicaciones**: Crear, ver, editar y eliminar publicaciones
- **Interacciones**: Sistema de "Me gusta" y "Guardados" en publicaciones
- **Notificaciones**: Los usuarios reciben notificaciones cuando alguien interactúa con sus publicaciones
- **API REST**: Acceso a toda la funcionalidad mediante API RESTful con autenticación

## Requisitos del sistema

- PHP 8.1 o superior
- Composer
- Laravel 10.x
- MySQL o MariaDB
- Node.js y npm para los assets

Laravel is accessible, powerful, and provides tools required for large, robust applications.
